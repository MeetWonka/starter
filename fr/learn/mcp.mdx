---
title: "Qu'est-ce que MCP (Model Context Protocol) ?"
description: "Comment les agents découvrent et utilisent les outils de manière sécurisée via un protocole standard."
---

<Info>
MCP est un protocole qui standardise comment un agent découvre les outils, demande des capacités et échange des données structurées avec des systèmes externes.
</Info>

### Concepts clés
- **Hôte** : exécute le LLM/agent
- **Serveur** : expose un catalogue d'outils avec schémas et autorisations
- **Client** : connecte l'agent au serveur
- **Outils** : fonctions typées que l'agent peut appeler (par exemple, recherche, IO fichier, API)

### Pourquoi MCP
- Découverte unifiée des outils disponibles
- Typage fort et métadonnées pour des appels plus sûrs
- Isolation et invites de permission pour les actions sensibles

### Exemple de descripteur de capacité

```json
{
  "tool": "search",
  "description": "Interroger les docs indexées",
  "inputSchema": { "type": "object", "properties": { "q": { "type": "string" } }, "required": ["q"] },
  "rateLimit": { "rpm": 60 }
}
```
